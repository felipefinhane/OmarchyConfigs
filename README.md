# OmarchyConfigs

My personal Arch Linux configuration files and dotfiles for a customized "Omarchy" setup.

## Overview

This repository contains my personal configuration files for Arch Linux, including dotfiles, system configurations, and setup scripts to create a personalized and efficient Linux environment.

## Structure

```
├── dotfiles/           # User configuration files
│   ├── shell/         # Shell configurations (bash, zsh, etc.)
│   ├── vim/           # Vim/Neovim configurations
│   ├── git/           # Git configurations
│   ├── x11/           # X11 and window manager configs
│   └── misc/          # Other miscellaneous dotfiles
├── system/            # System-level configurations
├── scripts/           # Setup and utility scripts
├── packages/          # Package lists and installation configs
└── README.md          # This file
```

## Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/felipefinhane/OmarchyConfigs.git
   cd OmarchyConfigs
   ```

2. Run the setup script:
   ```bash
   ./scripts/setup.sh
   ```

3. Or manually link specific configurations as needed.
